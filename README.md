# Aternos Keep-Alive Bot

A 24/7 bot that keeps your Aternos Minecraft server alive with a web dashboard.

## Features
- Web dashboard with server IP textbox
- Start / stop bot from the browser
- Auto reconnects if disconnected
- Live log viewer

## Setup
1. Clone this repo
2. Run `npm install`
3. Run `node index.js`
4. Open the dashboard in your browser
5. Type your Aternos server IP and hit Start!

## Hosting
Deploy to Railway.app for free 24/7 hosting.
Use UptimeRobot to ping the URL every 5 minutes.
